# Personal Portfolio

A modern engineering portfolio built with **React**, **TypeScript**, **Vite**, and **Tailwind CSS** to showcase software engineering projects, technical notes, and professional experience.

---

## Features

- Modern engineering-inspired UI
- Responsive design
- Smooth scrolling navigation
- Interactive project galleries
- Image & video support
- Engineering Notes section
- Resume preview
- Project architecture highlights
- Fully responsive layout

---

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### UI

- Responsive Layout
- Smooth Animations
- Modern Component Design
